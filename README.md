# Sistema de Compras

Este sistema foi desenvolvido como uma atividade acadêmica para a unidade curricular **Programação de Aplicativos**.

## 1- Configuração do Banco de Dados

* Abra o seu gerenciador de banco de dados (MySQL Workbench, DBeaver, etc.).
* Importe e execute o script SQL disponível na pasta `database/db_sistema_compras.sql`.

## 2- Configuração da Conexão

* Verifique a classe `jdbc.ClassConnection`.
* Se o seu MySQL exigir senha, altere a variável `PASSWORD` conforme necessário.

## 3- Execução

* Compile e execute a classe `src/main/java/applications/SistemaCompras.java`.
